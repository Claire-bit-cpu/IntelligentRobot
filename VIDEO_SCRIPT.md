# 智能机器人 (IntelligentRobot) 演示视频拍摄脚本

> **视频总时长**: 10~12 分钟  
> **目标观众**: 技术评审、企业决策者、开发者  
> **演示环境**: 飞书群聊 + IDEA 代码展示 + 终端命令行

---

## 🎬 准备工作

### 演示环境准备清单

- [ ] 飞书群聊（至少 2 个成员）
- [ ] 机器人已加入群聊
- [ ] 后端服务已启动（`java -jar IntelligentRobot-0.0.1-SNAPSHOT.jar`）
- [ ] 配置好环境变量（FEISHU_APP_ID, GITHUB_TOKEN 等）
- [ ] 准备一个测试用的 GitHub 仓库（用于演示 Git 集成）
- [ ] 屏幕录制软件就绪（建议 1080p，30fps）
- [ ] 麦克风测试完毕

## 🎬 拍摄技巧与注意事项

### 1. 屏幕录制设置

- **分辨率**: 1920x1080 (1080p)
- **帧率**: 30fps
- **码率**: 8-10 Mbps
- **格式**: MP4 (H.264)

### 2. 音频录制

- 使用专业麦克风（避免笔记本内置麦克风）
- 录制环境安静，无回声
- 语速适中（每分钟 200-250 字）
- 提前练习台词，避免口误

### 3. 剪辑建议

- 使用剪辑软件：Adobe Premiere Pro / Final Cut Pro / DaVinci Resolve
- 添加转场效果（淡入淡出、滑动等）
- 关键操作添加箭头、圆圈等高亮效果
- 添加背景音乐（注意版权）
- 添加字幕（中英文对照更好）

### 4. 常见问题处理

| 问题 | 解决方案 |
|------|---------|
| 演示时机器人无响应 | 提前测试，准备备用录屏 |
| 网络延迟导致响应慢 | 在本地环境演示，减少网络依赖 |
| 误操作 | 提前准备脚本，多次练习 |
| 时间超时 | 严格控制每个章节的时间，可删减非核心内容 |
